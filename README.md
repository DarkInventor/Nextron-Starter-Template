## Usage

### Create an App

```

# with pnpm
$ pnpm dlx create-nextron-app my-app --example with-tailwindcss
```

### Install Dependencies

```
$ cd my-app

# using pnpm
$ pnpm install --shamefully-hoist
```

### Use it

```
# development mode
$ pnpm run dev

# production build
$ `pnpm run build
```
